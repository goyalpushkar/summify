## Getting Started

### Prerequisites

-   Node.js
-   npm or yarn

### Installation

1.  Clone the repository:
```
bash
    git clone [repository-url]
    
```
2.  Navigate to the project directory:

```
bash
    cd server
    
```
3.  Install dependencies:
```
bash
    npm install
    
```
or
```
bash
    yarn install
    
```
### Running the Server

To start the server:
```
bash
npm run start
```
or
```
bash
yarn start
```
